# Venkata Rohith Pesala | Professional Portfolio

Welcome to my personal portfolio. This project showcases my journey, expertise, and projects as a **Full Stack Cloud Engineer**.

## Overview
A high-performance, modern portfolio built with:
- **Next.js 15** & **React**
- **Tailwind CSS** for styling
- **Framer Motion** for interactive animations

## Features
- **Interactive Experience Roadmap:** A detailed timeline of my career.
- **Dynamic Skills Grid:** Categorized technical expertise.
- **Education Section:** Academic achievements and research.
- **Glassmorphism UI:** Modern, clean, and responsive design.

## Local Setup
```bash
npm install
npm run dev
```
